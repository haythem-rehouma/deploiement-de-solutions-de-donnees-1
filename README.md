## Progression pédagogique

Le cours suit le cycle de vie complet d'une solution, des fondations jusqu'au projet intégrateur :

**Fondations** (01–03) → **Intégration / livraison continue** (04, 05, 10) →
**Conteneurs et orchestration** (06–09) → **Infrastructure et exploitation** (11–13) →
**Intégration finale** (14–15).
