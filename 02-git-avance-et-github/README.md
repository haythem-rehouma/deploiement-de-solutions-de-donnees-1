# Module 02 — Git avancé et GitHub

Deuxième module du cours **Développement et déploiement de solutions de données** (420-D30-BB).
Après les bases de Git (module 01), ce module passe à la **collaboration réelle** : organiser les branches, fusionner proprement, faire relire son code et contribuer en équipe via GitHub.

## Objectifs

À la fin de ce module, vous serez capable de :

- Choisir une **stratégie de branchement** adaptée (Git Flow, GitHub Flow, Trunk-Based).
- Distinguer et utiliser le **merge** et le **rebase**, et **résoudre les conflits**.
- Créer une **pull request**, mener une **revue de code** et la **fusionner**.
- Collaborer via **fork**, **clone**, **issues** et appliquer les **bonnes pratiques d'équipe**.
